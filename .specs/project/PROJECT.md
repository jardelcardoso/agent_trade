# Projeto: Agent Trade

## Visão
Construir um agente autônomo de trading com execução em modo dry run por padrão, armazenamento local em SQLite e orquestração via LLM.

## Objetivos
- Implementar pipeline de ingestão incremental de dados.
- Disponibilizar uma camada de risco e validação de ordens.
- Preparar a base para integração com corretoras futuras.

## Critérios de sucesso
- O projeto sobe com um comando simples.
- O fluxo principal pode ser executado em modo simulado.
- A estrutura de código está organizada por responsabilidade.
